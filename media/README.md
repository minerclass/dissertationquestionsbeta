# Dissertation Media Assets

This folder stores small public video assets used by Micah Miner's dissertation and research-facing web materials.

## Videos

- `dissertation-stages-of-consciousness.mp4`
- `ai-created-research-overview.mp4`
- `Designing Pedagogical Friction in the Age of Generative AI` is embedded from its public Google Drive source to avoid placing an 88.6 MB binary in Git history.

## Public page

These files are public-facing media assets only. Do not place participant data, interview materials, or IRB-protected research data in this folder.

# Final Qualifying Paper Review Companion

A public, static GitHub Pages companion site for Micah Miner's final qualifying paper, **Beyond Secondary Orality: Tertiary Algorithmicity and the Case for Pedagogical Friction**.

The site is designed for committee/advisor review and presents the full manuscript in a web-native format with:

- Sticky navigation and reading progress
- Argument-spine overview
- Committee review panels
- Searchable manuscript sections
- Searchable references
- Print-friendly review mode

## Publishing

This project is intended to publish from the repository root on GitHub Pages.

Expected live URL:

`https://minerclass.github.io/qualifying-paper-final-review/`

## Privacy note

This is a public-facing review artifact. It does not collect responses, include analytics, store participant data, or require a backend.

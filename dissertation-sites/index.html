Researcher Note

Use this page as the public-facing map of the dissertation web ecosystem. It is safe for public links, descriptions, and review navigation. It is not a place for participant uploads, interview transcripts, audio files, coded data, or anything private.

Suggested Reading Path

  1. Start with the qualifying paper resource hub to anchor the theoretical frame.
  2. Move through the literature review and companion sites to see the scholarly lineage.
  3. Open the mixed methods plan to connect the framework to the empirical study.
  4. Use the instrument suite only for reviewed, IRB-aligned participant workflows.

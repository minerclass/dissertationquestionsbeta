# Tertiary Algorithmicity Companion

An immersive public companion page for Micah Miner's qualifying paper, **Beyond Secondary Orality**.

This page is designed as a visual and conceptual guide to the paper's core argument:

- Ong's media ecology and the long arc of mediated consciousness
- Algorithmic secondary orality
- Tertiary algorithmicity
- Noetic, rhetorical, and existential displacement
- Pedagogical friction as an educational response
- The bridge from the qualifying paper to the dissertation
